Concept of WWW

  • It Stands for World Wide Web.
  • It is basically a system of internet servers that support specially formatted documents.
  • It is a collection of text pages,vedios,digital photographs and animations we can acess over the internet.
  • User access the World Wide Web facilities via a client called web browser.

HTTP Protocol

  1. Http stands for Hypertext Transfer Protocol.
  2. HTTP is based on the client-server architecture model ans a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
